# Changelog

## [1.0.0] - Adaptação para PWA Completo

### Adicionado
- ✅ Estrutura de monorepo com `apps/web` e `apps/api`
- ✅ PWA completo com manifest, service worker e instalação
- ✅ Backend API Node/Express com endpoints REST
- ✅ Dockerfiles para web (Nginx) e API
- ✅ Docker Compose para orquestração dos serviços
- ✅ Testes unitários com Vitest
- ✅ Testes E2E com Playwright
- ✅ CI/CD completo no GitHub Actions
- ✅ Deploy automático no GitHub Pages
- ✅ Documentação completa no README

### Modificado
- Front-end convertido de extensão Chrome para PWA web
- Integração com API backend para sincronização
- Fallback para localStorage quando offline
- Estrutura de projeto reorganizada

### Tecnologias
- Front-end: Vite, Vanilla JS, PWA Plugin
- Backend: Node.js, Express
- Testes: Vitest, Playwright
- Containerização: Docker, Docker Compose
- CI/CD: GitHub Actions
